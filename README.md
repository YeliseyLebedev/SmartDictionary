# Умный словарь 

Консольное приложение на C++ для работы с частотным словарём на основе красно-чёрного дерева.  
Программа загружает текст, подсчитывает частоту слов и сохраняет результат в файл.

## Возможности

- Загрузка текста из файла `dictionary.txt`
- Подсчёт частоты каждого слова
- Хранение данных в сбалансированном красно-чёрном дереве
- Вывод статистики в консоль
- Экспорт результатов в файл

## Технологии

- C++ — основной язык
- Windows Forms — графический интерфейс (для взаимодействия)
- RBTree — самописная реализация красно-чёрного дерева
